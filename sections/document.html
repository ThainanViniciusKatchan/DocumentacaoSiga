Documento

A parte de documento é onde a mágica acontece, até o momento criamos somente formulários estáticos e dinâmicos mas agora, vamos mostrar as informações obtidas atravez do formulário em um documento criado com HTML e FreeMarker, para a elaboração dos documentos não necessário começar um HTML do zero, pois as tags <html>, <body> e <head> já estão presentes de forma oculta no sistema, dessa forma, basta somente utilizar as tags de conteúdo e personalização.

O siga entende praticamente todas as tags do HTML dentro do documento, mas com CSS o assunto é outro, nem todos os parâmetgros são aceitos podendo muitas vezes ser completamente ignorado, na maioria dos casos o sistema ignora ajustes de margem em conteúdos quwe veem por padrão no sistema, como o tipo do documento.
